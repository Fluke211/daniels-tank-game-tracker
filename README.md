# Daniel's Tank Game — Progress Tracker

Public status page for **daniels-tank-game** (the game code itself stays private).

- **Live page:** https://fluke211.github.io/daniels-tank-game-tracker/
- `index.html` — the progress board (reads `progress.json`)
- `guide.html` — how the FORGE build system works, plus laptop setup
- `progress.json` — milestone/status data, mirrored automatically from the game
  repo by a small publisher on the server (the agent only edits it in the game repo)

Built with FORGE 🔥
